# Clinic Reservation System
_______

## Installation 
___

1. Composer installer
2. Xampp
3. Nodejs

___

## Commands
1- Composer Install
2- npm install

## How to run
Open 2 terminals in your text editor run **php artisan serve** and **npm run dev** in another then run xampp.
